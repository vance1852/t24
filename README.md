# auction-settle

二手实验室设备拍卖结算与审计工具 - 本地命令行工具，用于处理拍卖结算、费用计算和争议审计。

## 功能特性

- 📊 **完整的结算流程**：从出价到成交的全流程处理
- 💰 **精确的费用计算**：使用 Decimal 进行金额计算，无浮点数误差
- 🔍 **智能异常检测**：自动识别可疑出价、价格异常、重复序列号等问题
- 📝 **可追溯的解释**：每个拍品的成交过程可追溯，可解释
- ⚖️ **争议处理**：关联争议记录，自动给出处理建议
- 📤 **多格式输出**：支持表格、JSON、CSV 三种输出格式
- 🔒 **纯本地运行**：无需联网，无需数据库，数据全部存放在本地文件

## 快速开始

### 1. 初始化样例数据

```bash
auction-settle init --data-dir ./data
```

样例数据包含：
- 10 个拍品（显微镜、离心机、示波器、恒温箱、色谱仪、超低温冰箱、分析天平、切片机、超纯水系统等）
- 27 条出价记录（包含各种边界情况）
- 5 位买家（机构、企业、个人）
- 3 位卖家
- 6 条争议记录
- 可配置的费用规则

**边界情况内置：**
- 同一买家连续加价（短时间内多次出价）
- 撤回出价（BID008 被撤回）
- 出价时间相同（L1007 的两个出价时间完全相同）
- 低于保留价（多个出价低于保留价被排除）
- 买家保证金不足（高风险买家保证金不足）
- 拍品被管理员撤拍（L1008 已撤拍）
- 个人买家限制（L1005 限制个人买家）
- 重复序列号（L1001 和 L1010 使用相同序列号）
- 冷链运输要求（L1006 超低温冰箱需要冷链）

## 成交规则

1. **最高价成交**：同一拍品按有效最高价成交
2. **时间优先**：出价相同时，更早出价优先
3. **撤回排除**：撤回过的出价不参与成交
4. **保留价规则**：低于保留价不成交
5. **保证金规则**：高风险买家保证金不足不能赢得拍品（需 ≥ 保留价的 20%）
6. **撤拍排除**：被撤拍的拍品从结算中排除
7. **个人限制**：限制个人买家的拍品，个人买家不能成交

## 费用规则（可配置）

| 费用项目 | 规则 | 默认值 |
|---------|------|--------|
| 平台佣金 | 成交价 × 比例 | 5% |
| 检测服务费 | 成交价 × 比例 | 2% |
| 包装费 | 固定金额 | ¥50 |
| 冷链费 | 固定金额（需冷链时收取） | ¥200 |
| 木箱费 | 固定金额（需木箱时收取） | ¥150 |
| 省内物流费 | 固定金额 | ¥100 |
| 跨省物流费 | 固定金额 | ¥300 |
| 拆单附加费 | 固定金额（支持拆单时收取） | ¥50 |
| 税金 | （成交价 + 相关费用）× 税率 | 13% |
| 卖家撤拍罚金 | 固定金额 | ¥200 |

## 争议处理建议

系统会根据争议类型自动给出处理建议：

| 建议状态 | 含义 | 适用场景 |
|---------|------|---------|
| `AUTO_DISMISSED` | 可自动驳回 | 撤回出价争议、保留价争议（处理正确时） |
| `NEEDS_REVIEW` | 需要人工复核 | 可疑出价、价格异常、重复序列号等 |
| `REFUND_SUGGESTED` | 建议退款 | 保证金不足但已成交等情况 |

## 命令说明

### 全局选项

| 选项 | 说明 |
|------|------|
| `--data-dir PATH` | 指定数据目录路径 |
| `--help` | 显示帮助信息 |

### 所有命令

| 命令 | 说明 |
|------|------|
| `init` | 初始化样例数据 |
| `settle` | 计算批次成交结果 |
| `buyer-bill` | 生成买家应付账单 |
| `seller-statement` | 生成卖家结算单 |
| `audit` | 检查异常和争议 |
| `explain lot <LOT_ID>` | 解释拍品成交过程 |
| `export` | 导出汇总报表 |

### 命令通用选项

| 选项 | 说明 |
|------|------|
| `--output-format [table\|json\|csv]` | 输出格式（默认 table） |
| `--output PATH` | 输出文件路径（JSON/CSV 格式时使用） |
